# Food Security Data Mining

## Project Overview
This project focuses on analyzing the Global Food Security Index 2019 dataset using data mining and machine learning techniques.

The study applies the Knowledge Discovery in Databases (KDD) process, including:
- Data preprocessing and cleaning
- Data visualization and exploration
- Classification analysis
- Clustering analysis
- Model evaluation and interpretation

## Dataset
Global Food Security Index 2019 Dataset

## Objectives
- To analyze global food security patterns
- To classify food security levels using machine learning
- To identify clusters of countries with similar food security characteristics

## Data Mining Techniques
- Random Forest Classification
- K-Means Clustering

## Tools and Technologies
- Python
- Google Colab
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
